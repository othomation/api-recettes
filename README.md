## Description

API 'recettes' for create, modify, delete and view recipes.

## What is used here ?

NestJS, TypeORM, MySQL. (also Swagger)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
